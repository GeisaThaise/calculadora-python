# Calculadora em Python

## Sobre o projeto
Este projeto é uma calculadora desenvolvida em Python para realizar operações matemáticas básicas pelo terminal.

Foi criado com o objetivo de praticar lógica de programação e versionamento com Git e GitHub.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão

## Como executar
Abra o terminal na pasta do projeto e execute o arquivo Python.

## Tecnologias utilizadas
- Python
- Git
- GitHub
